PASO 2️⃣: DESPLEGAR FRONTEND
────────────────────────────

1. Ve a https://vercel.com/dashboard
2. Click "Add New" → "Project"
3. Selecciona el repositorio "gestion-despachos" NUEVAMENTE
4. Configura:
   • Framework: Create React App
   • Carpeta raíz: . (punto, significa raíz)
   • Build command: npm run build
   • Start command: npm start (o similar)

5. Agrega VARIABLE DE ENTORNO:
   
   REACT_APP_API_URL = https://[TU-BACKEND-VERCEL]/api
   
   Ejemplo:
   REACT_APP_API_URL = https://mi-backend.vercel.app/api

6. Click "Deploy"
7. ⏳ Espera 2-3 minutos
8. 🎉 ¡Tu app estará en línea!

🆘 SOLUCIÓN DE PROBLEMAS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

❌ "Cannot connect to database"
   → Verifica que DATABASE_URL está correcta en Vercel
   → Comprueba que Render está activo
   → Reinicia el deployment en Vercel

❌ "Frontend muestra errores de conexión"
   → Verifica que REACT_APP_API_URL apunta al backend correcto
   → Asegúrate de que la URL del backend fue copiada correctamente
   → Espera 1-2 minutos después del deployment

❌ "Build falló"
   → Revisa los logs en Vercel Dashboard
   → Asegúrate de que la carpeta raíz es correcta
   → Intenta redesplegando
